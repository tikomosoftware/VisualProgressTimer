AIエージェント用：アプリ開発仕様書案

1. プロジェクト概要
アプリ名: Visual Progress Timer

目的: タイムタイマーの「残り時間が色面で減っていく」視覚的効果を再現したWindowsデスクトップアプリ。

技術スタック: 指定なし（推奨： C# / WPF）

※軽量かつデスクトップでの描画パフォーマンスを重視。

2. コア機能 (Core Features)
タイマー設定:

円形（または扇形）のゲージをドラッグして直感的に時間を設定。

最大60分（設定で最大時間の変更も可能にする）。

カウントダウン制御:

スタート / ストップ / リセット機能。

残り時間に合わせて、色面が時計回りに消えていく（または反時計回りに減る）アニメーション。

通知機能:

タイムアップ時にシステム通知およびアラーム音の再生。

3. 外観・カスタマイズ (Appearance & UI)
色変更機能 (Color Themes):

ゲージの色: ユーザーがカラーピッカー、またはプリセット（赤、青、緑、オレンジ）から選択可能。

背景色: ダークモード / ライトモード対応。

ウィンドウ制御:

最前面表示 (Always on Top): 他の作業中も常に視界に入るように切り替え可能。

透過設定: ウィンドウ自体の不透明度を調整し、作業を邪魔しないようにする。

枠なしモード: 最小限の円形タイマーだけを表示するフローティングモード。

4. UI/UX 仕様
レスポンシブ: ウィンドウサイズに合わせてタイマーの円がリサイズされること。

操作性: マウスホイールでの時間微調整機能。

5. 実装ステップの指示（AIへのプロンプト例）
「上記仕様に基づき、まずはメインの円形タイマーを描画し、カウントダウンに合わせて色面が減っていく基本ロジックを実装してください。その後、カラーピッカーによる色変更機能と、Windowsの『最前面表示』機能を順次追加してください。」
